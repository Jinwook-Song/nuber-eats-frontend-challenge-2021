- 이전의 챌린지 과제였던 로그인 파트와 크게 다르지 않아서 무난하게 하실 수 있는 챌린지 과제입니다. tailwind와 react hook form, 쿼리어, apollo client가 아직 익숙치 않으신 분들은 많은 연습을 하실 수 있는 과제입니다.

- useForm의 formState를 이용하여 form의 입력이 valid한지 테스트 해보세요.
